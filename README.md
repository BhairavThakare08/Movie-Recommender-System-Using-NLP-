# 🎬 Movie Recommendation System (NLP + FastAPI + Streamlit)

## 🚀 Overview

This project is a **Content-Based Movie Recommendation System** built using **Natural Language Processing (NLP)** techniques.
It suggests movies based on similarity by analyzing metadata like genres, keywords, and descriptions.

---

## ✨ Key Features

* 🔍 Smart movie search with autocomplete
* 🎯 Content-based recommendations (TF-IDF + Cosine Similarity)
* 🧠 NLP-driven similarity engine
* 🌐 FastAPI backend for scalable APIs
* 🎨 Streamlit UI for interactive experience
* 🎥 TMDB API integration for posters & movie details
* ⚡ Real-time recommendations

---

## 🧠 How It Works

1. Movie metadata is preprocessed
2. Text data is converted using **TF-IDF Vectorization**
3. Cosine similarity is computed
4. Similar movies are ranked and returned

---

## 🏗️ Tech Stack

**Machine Learning / NLP**

* Python
* Pandas, NumPy
* Scikit-learn (TF-IDF, Cosine Similarity)
